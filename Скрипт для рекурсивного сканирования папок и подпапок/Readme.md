Что это такое?
Скрипт на PowerShell для рекурсивного сканирования папок и подпапок, который
показывает объём оных. Главным образом, нужен для выявления наиболее крупных
каталогов. Папки, размером меньше 99 мб не сканируются и не отображаются, т.к.
смысла в этом особого нет.

----------------------------------------------------------------------------------

Версия 4.1.
- Оптимизировано использование памяти и скорость работы для больших каталогов.
- Папки меньше 99 мб НЕ сканируются рекурсивно и не выводятся.
- Использован 1 проход по файловой системе для ускорения работы.
- Использован Dictionary для ускорения работы с кэшем.
- По тестам, версия 4.1 выдаёт примерно 15 сек. против 25-и у предыдущих.
- Также показывает скрытые и системные папки.

----------------------------------------------------------------------------------

Как пользоваться?
- Открываем PowerShell ISE от админа (в обычном PowerShell-e может не работать, кста).
- Переходим в нужный каталог. Если это Program Files, то его нужно будет писать
в кавычках, т.е. вот так: cd 'Program Files'
- И запускаем скрипт нажатием на F5. Всё.
