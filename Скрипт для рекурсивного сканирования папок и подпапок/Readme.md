Версия 4.1.
- Оптимизировано использование памяти скорость работы для больших каталогов.
- Папки меньше 99 мб НЕ сканируются рекурсивно и не выводятся.
- Использован 1 проход по файловой системе для ускорения работы.
- Использован Dictionary для ускорения работы с кэшем.
- По тестам, версия 4.1 выдаёт примерно 15 сек. против 25-и у предыдущих.
- Также показывает скрытые и системные папки.

----------------------------------------------------------------------------------

Как пользоваться?
- Открываем PowerShell ISE от админа.
- Переходим в нушный каталог. Если это Program Files, то его нужно будет писать
в кавычках, т.е. вот так: cd 'Program Files'
- И запускаем скрипт. Всё.
