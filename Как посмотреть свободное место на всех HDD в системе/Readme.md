Показывает метку диска, его полное название, сколько места всего, сколько осталось в GB и сколько осталось в процентах.

Запускается либо через PowerShell ISE, либо через обычный PowerShell и копирование кода в оный.
Права администратора для запуска не нужны.

Запустить данный скрипт на Windows XP и любых его итерациях, включая Windows Server 2003 нельзя, так как там нет PowerShell-a.
На всех остальных Виндах, включая серверные, начиная с 2008 R2 и заканчивая 2019 - всё ок.

-------------------------------------------------------------------------------------------------------------------------------

Если возникает ошибка при выполнении политики - разрешаем выполнение скрипта:

Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

или для постоянного изменения:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned

Хотя, всё должно работать и без этих строчек.
