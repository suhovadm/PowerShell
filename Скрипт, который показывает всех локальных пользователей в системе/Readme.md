Скрипт на PowerShell, который показывает всех локальных пользователей в системе,  
какие у них права, в каких группах они состоят, когда был последний вход в систему,  
включена ли учётка, установлен ли пароль на учётке. Админские учётки подсвечиваются зелёным.

* Группы указаны условно - если это админ, покажет группу "Administrators", если не админ - не покажет ничего.
