Запускается через PowerShell ISE, или с помощью копирования кода в обычный PowerShell.
Права администратора для запуска не нужны.
Показывает все локальные соединения, т.е. 127.0.0.1 и все внешние соединения, т.е. наш IP,
а самое главное - показывает порты, которые висят на внешнем соединении, т.е. что именно у нас открыто:
- SSH, YouTube, GitHub, Telegram, или что-то другое... Может быть, имеет смысл задуматься о своем поведении и карьере?
