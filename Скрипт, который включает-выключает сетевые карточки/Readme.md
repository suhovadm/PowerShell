Что это такое?  

Скрипт на PowerShell для управления сетевыми адаптерами.  
Версия 3.  
Позволяет быстро включать/выключать/перезапускать как физические,  
так и виртуальные сетевые карточки через консольное меню.  

Автоматическое определение:  
- активной сетевой карты  
- физических сетевых карт  
- виртуальных сетевых карт (VMware, Hyper-V, VPN и т.д.)  

Управление:  
- включение/выключение всех сетевых карт  
- управление только виртуальными сетевыми картами  
- управление только физическими сетевыми картами  
- быстрый перезапуск всех сетевых карт  

Защита от потери сети:  
- активная сетевая карта НЕ ОТКЛЮЧАЕТСЯ, чтобы избежать потери сети  

Удобный вывод:  
- имя адаптера  
- описание  
- статус  
- IP-адрес  
- MAC-адрес  
- скорость соединения  

Как пользоваться?  
Запускаем PowerShell ISE от админа, кидаем туда скрипт и работаем.  

PowerShell 5.1+  
Windows 10+ / Windows Server 2016+
