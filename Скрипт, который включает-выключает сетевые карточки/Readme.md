Что это такое?  

Скрипт на powershell для быстрого включения/выключения сетевых карточек - железных и виртуальных.  
С его помощью можно как просто посмотреть все сетевые  
карточки в системе, так и включить/выключить определённые девайсы.  
Также, скрипт показывает MAC-адреса, скорость линка и статус: up, или disconnect.  

Как пользоваться?  
Запускаем обычный PowerShell, или PowerShell ISE от админа, кидаем туда скрипт и работаем.  
Скрипт работает начиная с PowerShell 3.0 / Windows 8 / Windows Server 2012.  
Полностью совместим с PowerShell 7.
