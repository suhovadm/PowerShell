Что это такое?  

Калькулятор степеней на PowerShell.  

Версия 1.  

* Работает даже на старых версиях PowerShell, начиная с 2.0 / Windows 7.  
* Включена принудительная поддержка UTF-8, чтобы не было проблем с кодировкой.  
* Скрипт работает в бесконечном цикле, не надо ничего перезапускать.  
* Для выхода из цикла используется Ctrl + C, а также команды exit и quit.  
* При ошибке ввода используется continue, чтобы программа не падала.  
* Добавлена цветовая подсветка при ошибке, а также при выводе результата.  
* Очистка консоли производится командами cls и clear. Их можно вводить в любой строке.  
* Программа поддерживает вывод истории по команде history. История хранит 100 результатов. Параметр можно настроить.
