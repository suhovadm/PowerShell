Как посмотреть версию PowerShell ?

Способ #1, работает везде:

$PSVersionTable.PSVersion

-----------------------------------------------------

Способ #2, тоже самое, что предыдущий, но короче:

$host.Version

Способ #3, покажет вывод в более привычном виде:

Get-Host | Select-Object Version

-----------------------------------------------------
