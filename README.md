Как посмотреть версию PowerShell ?

Способ #1, самый лёгкий и простой:

$PSVersionTable

---------------------------------------------------------

Способ #2, работает везде:

$PSVersionTable.PSVersion

---------------------------------------------------------

Способ #3, тоже самое, что предыдущий, но короче:

$host.Version

---------------------------------------------------------

Способ #4, покажет вывод в более привычном виде:

Get-Host | Select-Object Version

---------------------------------------------------------

Способ #5, если надо посмотреть через командную строку:

powershell -Command "$PSVersionTable.PSVersion"

---------------------------------------------------------
